About UserPlus Tracking

Learn about the tracking capabilities and test environment

What is UserPlus?

UserPlus is a comprehensive user tracking solution that helps website owners understand user behavior, session information, and engagement metrics.

This test environment was created to demonstrate and validate the tracking capabilities of the UserPlus script, including:

  • Click activity tracking
  • User identification (anonymous vs. identified)
  • Session management
  • Custom event tracking
  • User state transitions (active, idle, inactive)
  • Page navigation tracking

Use the various pages in this test environment to validate all aspects of user tracking.

Technical Details

The UserPlus tracking script uses browser storage to maintain user and session information:

  • LocalStorage: Stores persistent user information such as user ID and type
  • SessionStorage: Stores temporary session information
  • Cookies: Used as a fallback for environments where localStorage is unavailable

The script tracks session states:

  • Active: User is actively engaging with the site
  • Idle: User has been inactive for some time (60 seconds)
  • Inactive: Session has been terminated or user has logged out

All event data is sent to the UserPlus API endpoint for analytics processing.

Implementation Guide

To implement UserPlus tracking on your website:

  1. Include the userplus.js script in your HTML
  2. Initialize the tracking with your script ID
  3. Use the provided methods to track user interactions:
// Initialize tracking
window.userplus.init("YOUR_SCRIPT_ID");
// Identify a user
window.userplus.identify("user_123", { name: "John Doe" });
// Track custom event
window.userplus.event("purchase", { product_id: "abc123" });

For more detailed implementation guides and API documentation, please refer to the official UserPlus documentation.
